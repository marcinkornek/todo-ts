# Project setup

## Requirements
- Node 10 or newer
- Yarn
- Xcode
- [Getting Started Â· React Native](https://facebook.github.io/react-native/docs/getting-started) - configuration for a project with native code

## Setup for development

1. Clone:
```
$ git clone git@github.com:marcinkornek/todo-ts.git
$ cd todo-ts
```

2. Install dependencies:
```
$ yarn
```

3. Install native dependencies:
```
$ cd ios && pod install && cd ..
```
